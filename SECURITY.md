# Security Policy

## Supported Versions

Please use following node js version

| Version | Supported          |
| ------- | ------------------ |
| 10.0.x  | :white_check_mark: |
| 12.0.x  | :white_check_mark: |
| 14.0.x  | :white_check_mark: |
| < 10.0  | :x:                |
